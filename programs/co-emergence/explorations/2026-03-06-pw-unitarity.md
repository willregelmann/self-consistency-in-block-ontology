# Is Measure-Theoretic PW Conditioning Approximately Unitary?

**Date:** 2026-03-06
**Program:** co-emergence
**Status:** Negative result — c-dependence is NOT approximately unitary in the toy model

## Question

The measure-theoretic conditioning (Definition 4.3, Propositions 4.4–4.6) produces c-dependent quantum states ρ_{S|c}. In standard Page-Wootters, the t-dependence is unitary: |ψ(t)⟩ = exp(−iH_S t)|ψ(0)⟩. Does the toy model's c-dependence approximate this structure?

Two specific tests:
1. **Approximate unitarity:** Are the magnitude profiles |φ_c(s)| approximately c-independent? (Unitary evolution preserves magnitudes.)
2. **Hamiltonian generation:** For d_clock ≥ 3, do the conditional states lie on a one-parameter unitary orbit? (Toeplitz Gram matrix, orbit consistency test.)

## Tests Performed

**Script:** `programs/co-emergence/tests/pw_unitarity_test.py`

### 1. Magnitude variation

Measured ||mag_c − mag_c'|| (Frobenius distance between magnitude profiles) for multiple configurations.

| dims    | d_clock | signature  | mag_var |
|---------|---------|------------|---------|
| (3,4)   | 3       | Lorentzian | 0.331   |
| (3,8)   | 3       | Lorentzian | 0.326   |
| (3,2,2) | 3       | Lorentzian | 0.332   |
| (4,4)   | 4       | Lorentzian | 0.361   |
| (4,2,2) | 4       | Lorentzian | 0.359   |
| (2,2,2) | 2       | Lorentzian | 0.341   |

**Verdict:** Magnitude variation ≈ 0.33 across all cases. The c-dependence is far from unitary.

### 2. Gram matrix Toeplitz property

For Hamiltonian-generated evolution, the Gram matrix G_{cc'} = ⟨φ_c|φ_c'⟩ should be Toeplitz (depend only on |c − c'|). Toeplitz deviations are small (~0.02), but this is misleading — the orbit test (below) fails badly.

### 3. Unitary orbit test (d_clock ≥ 3)

Construct the minimal-rotation unitary U mapping φ₀ → φ₁, then test whether U|φ_c⟩ = |φ_{c+1}⟩ for c ≥ 1.

| dims    | d_clock | orbit error (c=1→2) | orbit error (c=2→3) |
|---------|---------|---------------------|---------------------|
| (3,4)   | 3       | 0.955               | —                   |
| (3,8)   | 3       | 0.874               | —                   |
| (4,4)   | 4       | 1.005               | 0.866               |
| (4,2,2) | 4       | 1.021               | 0.866               |

**Verdict:** Orbit errors ≈ 1.0 (on the scale where √2 is maximum for normalized states). The c-dependence is NOT generated by a single Hamiltonian.

### 4. Phase linearity

If the c-dependence were Hamiltonian-generated, the phase of φ_c(s) relative to φ₀(s) would be linear in c: phase_c(s) = H(s) · c. Linearity errors are ~0.24, confirming non-linear c-dependence.

### 5. Weak coupling sweep

Swept α_clock from 0.5 to 0.001 with fixed dims = (3, 4):

|---------|---------|----------|-----------|---------|
| 0.500   | 0.331   | 0.021    | 0.955     | 0.241   |
| 0.200   | 0.331   | 0.021    | 1.052     | 0.256   |
| 0.100   | 0.331   | 0.021    | 1.093     | 0.262   |

**Verdict:** Reducing the marginal coupling has NO effect. α_clock is separable (depends only on c), so it contributes a c-dependent global phase but doesn't affect the clock-system interaction structure. The non-unitarity comes from the non-separable external field h_{(c,s)}.

### 6. Separable h test

With h_{(c,s)} = h^C_c + h^S_s (perfectly separable external field):

| interaction ε | mag_var | toep_dev | orbit_err | DM variation |
|---------------|---------|----------|-----------|--------------|
| 0.0           | 0.004   | 0.00001  | 1.09      | ~0.01        |
| 0.1           | 0.031   | 0.0003   | 1.11      | —            |
| 0.3           | 0.131   | 0.003    | 1.15      | —            |
| 1.0           | 0.294   | 0.001    | 1.71      | —            |

**Critical observation:** With separable h, the conditional DMs become nearly c-independent (distance ~0.01) — magnitude variation drops to ~0.004. But there is essentially NO time evolution. The orbit error remains ~1 even at ε = 0 because the residual c-dependence (from self-coupling β|ψ|²) is not Hamiltonian-structured; the orbit test correctly detects this even when the perturbations are tiny.

Adding non-separable interaction produces c-dependence (growing mag_var) but it is immediately non-unitary.

## Why This Fails

### Structural analysis

The fixed point has ψ*_σ ∝ exp((−1 + iθ)R_σ), so the conditional state is:

φ_c(s) ∝ exp((−1 + iθ)R_{(c,s)})

with |φ_c(s)| ∝ exp(−R_{(c,s)}) and arg(φ_c(s)) = θR_{(c,s)}.

**For c-dependence to be unitary, we need:**
1. |φ_c(s)| ≈ c-independent → requires R_{(c,s)} ≈ f(c) + g(s) (separable curvature)
2. Phase structure linear in c → requires R_{(c,s)} − R_{(c₀,s)} = c · V(s) (linear interaction)

**Why condition 1 fails:** R_{(c,s)} = h_{(c,s)} + α₁M₁(c) + Σ_{j>1} αⱼMⱼ(sⱼ) + β|ψ_{(c,s)}|². The external field h_{(c,s)} has generic (non-separable) dependence on both c and s. The self-coupling β|ψ|² is inherently non-separable. These terms make exp(−R_{(c,s)}) genuinely c-dependent.

**Why condition 2 fails:** Even if magnitudes were c-independent, the phase θR_{(c,s)} would need to be linear in c for fixed s. With generic h, R_{(c,s)} has no reason to be linear in c.

### Physical interpretation

In standard PW, the clock is assumed to be ideal: large, semiclassical, weakly coupled to the system. The clock provides a time label without significantly entangling with the system. Under these conditions, conditioning on the clock is approximately unitary.

In the toy model:
- d_clock = 2, 3, or 4 — these are extremely quantum clocks (far from semiclassical)
- The clock-system coupling through h is O(1) — the clock strongly back-reacts on the system
- There is no parameter regime that makes the clock semiclassical

The non-unitarity is expected and physically correct: a strongly quantum, strongly coupled clock does not provide a good time reference.

## What Would Be Needed

For the c-dependence to approximate unitary time evolution, the toy model would need:

1. **Large d_clock with fixed d_sys** (semiclassical clock limit): dims = (d_clock, d_sys) with d_clock >> d_sys. The clock's conditional state should become approximately constant as d_clock → ∞.

2. **Structured clock-system interaction:** The non-separable part of R_{(c,s)} would need to be approximately linear in c for fixed s. This would require a specific structure in h that mimics a Hamiltonian interaction — not generic random h.

3. **Weak clock-system coupling:** The interaction part of R should be small relative to the separable part, so that magnitudes are approximately c-independent while phases carry the c-dependence.

These conditions amount to engineering a toy model that explicitly contains a semiclassical clock — which would test whether self-consistency preserves the PW structure, but not whether self-consistency generates it.

## Implications for the Paper

The measure-theoretic conditioning (Definition 4.3) is strictly weaker than the Page-Wootters mechanism:

| Property | Measure conditioning | Standard PW |
|----------|---------------------|-------------|
| c-dependent quantum states | ✓ (Proposition 4.6) | ✓ |
| Mixture → partial trace | ✓ (Proposition 4.4) | ✓ |
| Riem = classical | ✓ (Proposition 4.5) | N/A |
| Unitary c-dependence | ✗ (this exploration) | ✓ (assumed) |
| Hamiltonian generator | ✗ (this exploration) | ✓ (assumed) |

The construction resolves the *algebraic* circularity (no total Hilbert space needed) but does not reproduce the *dynamical* content of PW (unitary time evolution). This distinction should be noted in the paper's Open Problem 2.

## Verdict

**Negative.** The toy model's c-dependence is quantum but not unitary, and not generated by a single Hamiltonian. No parameter regime tested produces approximate unitarity. The measure-theoretic conditioning produces the algebraic structure of PW (c-dependent quantum states, mixture consistency) but not the dynamical structure (unitary evolution). Closing this gap requires either a semiclassical clock limit (large d_clock, structured interaction) or a fundamentally different argument for how unitarity emerges from self-consistency.

This is a clean negative result that sharpens the open problem: the question is no longer "can PW be reformulated without Hilbert space?" (yes, Definition 4.3) but "under what conditions does self-consistency produce unitary c-dependence?"
