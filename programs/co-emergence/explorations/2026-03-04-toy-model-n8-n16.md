## Motivation

The N=4 toy model (Remark 8 in the paper) demonstrated that Lorentzian self-consistency produces quantum coherences absent in the Riemannian case. But complex off-diagonal density matrix elements are necessary, not sufficient, for quantum mechanics. Three structural tests require N > 4:

1. **Subsystem composition**: partial traces must compose — Tr_B(ρ_AB) = ρ_A
2. **Born rule**: Tr(ρ_A P_k) matches direct computation from ψ* in any basis
3. **Interference**: destructive interference distinguishes quantum from classical-with-phases

## Model

### Generalization from N=4

The model generalizes to arbitrary tensor product structures S = S_1 × S_2 × ... × S_k with dims = (d_1, ..., d_k) and N = Π d_j. The curvature is:

R_σ(ψ) = h_σ + Σ_j α_j · M_j(σ_j) + β · |ψ_σ|²

where M_j(i) = Σ_{σ: σ_j=i} |ψ_σ|² is the marginal of subsystem j. Each subsystem gets its own coupling α_j. The weight function and self-consistency map are unchanged in form.

### Configuration spaces tested

- **N=4:** S = 2×2 (regression, original parameters h=[1.0, 0.7, 0.5, 1.2])
- **N=8:** S = 2×2×2, h drawn from Uniform[0.5, 1.5] with seed 123, α = [0.5, 0.3, 0.4]
- **N=16:** S = 2×2×2×2, h drawn from Uniform[0.5, 1.5] with seed 456, α = [0.5, 0.3, 0.4, 0.35]

All models use β = 0.4, γ = -1 + i (Lorentzian) or γ = -1 (Riemannian).

## Results

### 1. Subsystem composition: PASS

All 6 pair→single reductions at N=8 and all tested reductions at N=16 satisfy ‖Tr_X(ρ_YX) - ρ_Y‖_F < 10⁻¹⁰. Composition holds for both Lorentzian and Riemannian fixed points.

**N=16 exclusive: three-level composition chain.** The chain ρ_ABC → ρ_AB → ρ_A = ρ_A (direct) passes for all 6 tested triple→pair→single paths. This tests composition at three levels of coarse-graining and cannot be done at N=8.

### 2. Born rule: PASS

For every subsystem at N=8 and N=16, in computational, Hadamard, and random unitary bases:

p_born(k) = (U ρ_A U†)_{kk} = Σ_{bc...} |(U ψ)_{k,bc...}|² = p_direct(k)

Agreement to < 10⁻¹² in all cases. The Born rule is an exact consequence of the density matrix structure, not an approximation.

### 3. Interference: PASS

**Coherence:** All Lorentzian fixed points at N=8 and N=16 have |Im(ρ[0,1])| > 10⁻⁶. All Riemannian fixed points have |Im(ρ[0,1])| < 10⁻¹⁰.

**Destructive interference:** For each Lorentzian fixed point, there exists a measurement basis where p_quantum(k) < p_incoherent(k), where p_incoherent uses only diagonal elements of ρ (ignoring coherences). This is the hallmark of genuine quantum interference — classical probability distributions with phases cannot produce it.

The destructive-interference basis is constructed analytically: for 2×2 ρ with off-diagonal element c = |c|e^{iφ}, the basis with rotation angle π/4 and phase φ + π produces destructive interference. This is guaranteed whenever Im(ρ[0,1]) ≠ 0.

### 4. Robustness: PASS

N=16 tested across 5 random seeds for h. All show Lorentzian coherence in at least one subsystem. The result is not an artifact of a particular choice of external field.

## Interpretation

The self-consistency mechanism does not merely produce complex numbers in density matrices — it produces the full algebraic structure of quantum subsystems:

1. **Composition** means the subsystem structure is consistent: coarse-graining in different orders gives the same answer. This is a necessary condition for any theory that claims to derive Hilbert space structure.

2. **Born rule** is automatic given L²-normalized states and density matrices via partial trace. This is not a new postulate but a consequence of the mathematical structure. The point is that this structure emerges from self-consistency rather than being assumed.

3. **Destructive interference** is the critical test that separates quantum from classical-with-phases. A classical probability distribution with complex phases attached could have complex off-diagonal density matrix elements but would not exhibit destructive interference in measurement probabilities. The Lorentzian fixed points do.

## Caveats

1. **Born rule is kinematic, not dynamical.** The Born rule follows from the density matrix formalism once you have L²-normalized amplitudes and partial traces. What the toy model shows is that self-consistency produces the right *input* (complex amplitudes with the right normalization), not that it explains why measurement outcomes follow the Born rule. The measurement problem remains.

2. **All subsystems are 2-dimensional.** The tests use dims = (2,2,2,...). Testing with mixed dimensions (e.g., 2×3×2) would strengthen the result but is not done here.

3. **No dynamics.** The model finds fixed points; it does not evolve states. Whether unitary time evolution emerges from self-consistency is a separate (harder) question.

4. **Finite N.** The toy model has N = 4, 8, or 16 smooth structures. The physical case has a continuum. Scaling behavior beyond N=16 is untested.

## Open questions for future work

1. **Mixed-dimension subsystems:** Does composition hold for 2×3 or 2×3×4 structures?
2. **Entanglement measures:** Do the subsystem density matrices show entanglement entropy scaling consistent with known results?
3. **Unitary structure:** Is there a sense in which the self-consistency map generates unitary evolution on subsystems?
4. **Continuum limit:** What happens as N → ∞?
